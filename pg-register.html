Screenshot
Details

This is the home page. Home page will list down the most recent 9 quizzes on homepage. This means any quizzes that have the most recent modified on date in database will be listed here. If there are more than 9 quizzes in database, home page will also show pagination.

Each quiz list item will have quiz title, a description of maximum 300 characters, number of questions and a direct link to attempt quiz. If the quiz type is of image, it will show image icon in quiz header.

Home page also have a basic description of the Quizbee Project above the quizzes list which can be modified from Home controller in Index action. Links to creating a new quiz and sharing on Facebook and Twitter and see morequizzes are also there.
